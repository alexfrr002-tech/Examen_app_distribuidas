1. Importa db.sql en phpMyAdmin.
2. Copia xampp-htdocs/af_sneakers_gateway dentro de la carpeta htdocs de XAMPP.
3. Entra a node-api y ejecuta: npm install
4. En una terminal: node backend.js
5. En otra terminal: node gateway.js
6. Abre la página en:
   - http://localhost/af_sneakers_gateway/
   - o si Apache usa 8080: http://localhost:8080/af_sneakers_gateway/
7. El backend queda en http://localhost:3001
8. El gateway queda en http://localhost:8081
